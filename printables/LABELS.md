# Station Labels (Parallel Branches)

Print and cut. Place next to each QR code.

---

## FOOD Branch (Print on RED paper)

```
+----------------------------------+
|  FOOD - STEP 1                   |
|  Location: FRIDGE                |
|  Scan to begin this branch!      |
+----------------------------------+
```

```
+----------------------------------+
|  FOOD - STEP 2                   |
|  Location: SNACK TABLE           |
|  (Unlocks after Step 1 solved)   |
+----------------------------------+
```

```
+----------------------------------+
|  FOOD - STEP 3                   |
|  Location: DRINK COOLER          |
|  Final step! Earn digits [4,1]   |
+----------------------------------+
```
